📝 BƯỚC 1: TẠO FILE .env

Tạo file mới tên ".env" (KHÔNG CÓ .txt) trong thư mục dự án:
c:\Users\trang\OneDrive\Desktop\RAG GYM\.env

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 BƯỚC 2: COPY NỘI DUNG SAU VÀO FILE .env:

SUPABASE_URL=your_supabase_url_here
SUPABASE_KEY=your_supabase_anon_key_here

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🌐 BƯỚC 3: ĐĂNG KÝ VÀ LẤY API KEYS TỪ SUPABASE

1. Vào: https://supabase.com
2. Click "Start your project" (MIỄN PHÍ!)
3. Đăng nhập bằng GitHub
4. Click "New project"
5. Điền thông tin:
   - Name: gym-rag-ai (hoặc tên bạn thích)
   - Database Password: tạo password mạnh (LƯU LẠI!)
   - Region: Southeast Asia (Singapore)
   - Click "Create new project"
6. Đợi 2-3 phút để Supabase tạo database

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🗄️ BƯỚC 4: TẠO CÁC BẢNG DATABASE

1. Trong Supabase dashboard, click "SQL Editor" (menu bên trái)
2. Click "New query"
3. Copy toàn bộ code SQL từ file: SUPABASE_SQL_SCHEMA.txt
4. Paste vào SQL Editor
5. Click "Run" (hoặc Ctrl+Enter)
6. Đợi chạy xong (sẽ thấy "Success. No rows returned")

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔑 BƯỚC 5: LẤY API KEYS

1. Click "Settings" ⚙️ (menu bên trái, phía dưới)
2. Click "API" trong menu Settings
3. Bạn sẽ thấy:
   
   📍 Project URL:
   https://xyzcompany.supabase.co
   👆 Copy URL này!
   
   📍 Project API keys:
   - anon public: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
   👆 Copy key "anon public" này!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✏️ BƯỚC 6: ĐIỀN VÀO FILE .env

Mở file .env và thay thế:

SUPABASE_URL=https://xyzcompany.supabase.co
SUPABASE_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

(Thay bằng URL và KEY thật của bạn!)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💾 BƯỚC 7: LƯU FILE VÀ CHẠY LẠI APP

Lưu file .env, sau đó chạy:

python app.py
