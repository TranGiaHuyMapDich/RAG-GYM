⚙️ SETUP SUPABASE (TÙY CHỌN):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Nếu KHÔNG setup Supabase:
  ✅ App vẫn chạy bình thường
  ❌ Không lưu data
  ⚠️ Hiện: "Chưa đăng nhập (dữ liệu không được lưu)"

Nếu MUỐN setup Supabase:
  1. Tạo project: https://supabase.com
  2. Run SQL từ: QUICK_SUPABASE_SETUP.txt
  3. Tạo .env:
     SUPABASE_URL=your-url
     SUPABASE_KEY=your-key
  4. Install:
     python -m pip install supabase python-dotenv
  5. Run: python app.py
  6. Thấy: ✅ Kết nối Supabase thành công!
