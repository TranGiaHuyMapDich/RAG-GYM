NẾU MUỐN LƯU DATA THẬT:

1. Vào: https://supabase.com
   → Sign up / Login
   → New Project
   → Đợi ~2 phút project khởi tạo

2. Tạo tables:
   → Vào SQL Editor
   → Copy code từ file: QUICK_SUPABASE_SETUP.txt
   → Paste và Run

3. Lấy API keys:
   → Settings → API
   → Copy: Project URL và anon public key

4. Tạo file .env trong thư mục project:
   
   SUPABASE_URL=https://your-project.supabase.co
   SUPABASE_KEY=your-anon-key-here

5. Cài thêm packages:
   
   python -m pip install supabase python-dotenv

6. Restart app:
   
   python app.py
   
   → Thấy: ✅ Kết nối Supabase thành công!

✅ XONG! Data giờ sẽ được lưu thật!

❓ FAQ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: App có chạy không có Supabase được không?
A: CÓ! App vẫn chạy đầy đủ, chỉ không lưu data.

Q: Tôi phải setup Supabase không?
A: KHÔNG bắt buộc. Chỉ cần nếu muốn lưu data thật.

Q: Làm sao biết đã đăng nhập?
A: Xem header bar: ✅ Đã đăng nhập (màu xanh)

Q: Data lưu ở đâu?
A: Nếu có Supabase: Lưu ở Supabase database
   Nếu không: Chỉ lưu user_id trong localStorage (browser)

Q: Tôi mất data khi refresh?
A: KHÔNG. User_id lưu trong localStorage, persistent.

Q: Test page là gì?
A: Trang test riêng với UI đơn giản để test tất cả APIs.

Q: Làm sao clear data?
A: F12 → Console → localStorage.clear() → Reload
