  Q: Lần đầu chạy mất bao lâu?
  A: 5-10 phút để tải AI models (~500MB). Lần sau rất nhanh!

  Q: Cần internet không?
  A: Chỉ lần đầu để tải models. Sau đó dùng offline được.

  Q: Cần bao nhiêu RAM?
  A: Tối thiểu 4GB RAM, khuyến nghị 8GB.

  Q: Dataset ở đâu?
  A: File "megaGymDataset.csv" phải cùng thư mục với app.py

  Q: Làm sao dừng server?
  A: Nhấn Ctrl+C trong terminal

  💪 TĂNG CƠ (Muscle Gain)
     - Tập nặng, reps thấp
     - Ăn thặng dư calo
     - Progressive overload

  🔥 GIẢM CÂN (Weight Loss)
     - Nhiều cardio + HIIT
     - Deficit calo
     - Protein cao

  ⚡ TĂNG SỨC MẠNH (Strength)
     - Tập rất nặng (3-5 reps)
     - Compound exercises
     - Nghỉ dài giữa sets

  🏃 TĂNG SỨC BỀN (Endurance)
     - Reps cao (15-20)
     - Nghỉ ngắn
     - Circuit training

  ECTOMORPH (Người gầy)

  ├─ Đặc điểm: Dễ tăng cơ và giảm mỡ
  ├─ Tập: Cân bằng cardio + strength
  └─ Ăn: Balanced diet

  ENDOMORPH (Người dễ béo)
  ├─ Đặc điểm: Dễ tích mỡ
  ├─ Tập: Nhiều cardio + HIIT
  └─ Ăn: Kiểm soát calo

┌──────────────────────────────────────────────────────────────────────┐
│  🎓 HỌC THÊM                                                         │
└──────────────────────────────────────────────────────────────────────┘

  📚 RAG (Retrieval Augmented Generation):
     - Kết hợp tìm kiếm + AI generation
     - Semantic search với embeddings
     - Vector database (ChromaDB)

  🧠 Sentence Transformers:
     - Chuyển text → vector (768 dimensions)
     - Hỗ trợ đa ngôn ngữ
     - Model: paraphrase-multilingual-mpnet-base-v2

  ⚡ FastAPI:
     - Modern Python web framework
     - Async support
     - Automatic API docs

┌──────────────────────────────────────────────────────────────────────┐
│  💡 TIPS & TRICKS                                                    │
└──────────────────────────────────────────────────────────────────────┘

  1. Câu hỏi cụ thể → Kết quả tốt hơn
     ✅ "Bài tập ngực với tạ đơn cho người mới"

  2. Chọn đúng loại cơ thể → Kế hoạch phù hợp hơn

  3. Điền chiều cao/cân nặng → Nhận tính BMI

  4. Chọn thiết bị có sẵn → Bài tập thực tế hơn

  5. Tăng ngày tập dần dần: 3 → 4 → 5 → 6 ngày/tuần
